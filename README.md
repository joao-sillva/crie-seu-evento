## Sobre o projeto
![preview](.github/preview.png)

Este projeto consiste em uma interface web simples, construída como parte da trilha Explorer oferecida pela Rocketseat.
O objetivo principal é abordar conceitos fundamentais dos tipos de campos HTML disponíveis para formulários. Durante as aulas, 
foram aplicados os seguintes conceitos:


Conceitos aplicados
* Tipos de campos HTML para formulários e sua utilização
* Estruturação e organização de campos com fieldset e legend
* Personalização de elementos como select e checkbox para melhorar usabilidade e acessibilidade
* Aplicação de técnicas de CSS para criar layouts responsivos e esteticamente agradáveis em formulários


## Tecnologias
